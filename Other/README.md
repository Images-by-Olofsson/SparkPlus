# Other - Private Workspace

This directory is a personal workspace for experimental files, temporary notes, and test data. Contents are excluded from version control via `.gitignore` (except `.gitkeep` and this file).

## Usage

Place any files here that should not be distributed or shared publicly. Examples include:

- **`experiments/`** -- Experimental code and prototypes.
- **`notes.md`** -- Personal notes and observations.

Files in this directory will not appear in the remote repository.
