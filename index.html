Chromium Browser Extension

SparkPlus

A browser extension that integrates directly with the Spark ticketing API, delivering real-time ticket monitoring, status change notifications, SLA tracking, and time reporting from the browser toolbar.

Download Latest Loading...

Features

Real-Time Notifications

Receive instant browser notifications when tickets are assigned, updated, or when status changes occur. Powered by direct Spark API integration with automatic token management and 50-minute refresh cycles.

Ticket Overview

View all your assigned tickets at a glance directly from the browser toolbar. The extension queries the Spark API in real time, filtering out resolved and closed statuses to show only actionable items.

Quick Actions

Perform common ticket operations, time reporting, and SLA tracking directly from the extension popup without switching context or navigating away from your current workflow.

Configurable

Customise notification filters, refresh intervals, display preferences, and excluded ticket statuses. The extension adapts to your workflow with granular control over every aspect of its behaviour.

Installation

1

Download

Click the Download Latest button above to get the latest PROD-READY-SETUP.zip package.

2

Extract

Unzip the downloaded file to a permanent location on your computer. Do not place it in a temporary or downloads folder.

3

Enable Developer Mode

Open chrome://extensions (or edge://extensions for Edge) in your Chromium-based browser and enable Developer mode in the top-right corner.

4

Load Extension

Click Load unpacked and select the extracted folder. SparkPlus will appear in your toolbar. Sign in to spark.iver.com to activate the API connection.
